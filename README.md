# 🔥 Prince Hacker Site

A Flask-based web application with fun mini games and a hacker-style UI.  
Built and deployed completely by me as a Class 11 student 🚀

## 🌐 Live Website
👉 https://prince-hacker-site.onrender.com

> ⚠️ Note: The site is hosted on Render (free plan), so the first load may take some time.

---

## 🎮 Features

- 🎯 Number Guessing Game  
- 🦖 Dino Game (custom JavaScript, not Google embed)  
- 🧠 Clean navigation between pages  
- ⚡ Hacker-style simple UI  
- 🔁 Restart & score handling in games  

---

## 🛠 Tech Stack

- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS, JavaScript  
- **Hosting:** Render  
- **Version Control:** Git & GitHub
